# Windows Packaging Assets

This folder holds Windows-specific branding files used by both PyInstaller and Inno Setup.

- `app.ico` is the current shared app/installer icon.
- `wizard-image.bmp` is an optional future large installer sidebar image.
- `wizard-small.bmp` is an optional future small installer header image.

The current `app.ico` is derived from an existing Dipsy sprite so packaging can ship with normal Windows metadata before dedicated branding art exists.
