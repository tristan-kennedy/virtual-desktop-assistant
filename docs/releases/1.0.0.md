# Dipsy Dolphin 1.0.0

## Highlights

- Dipsy now appears as a sprite-based desktop character with authored `idle` and `talk` animation instead of procedural drawing.
- Chat runs against a bundled local GGUF model through llama.cpp, with visible bubble dialogue and optional retro Windows speech.
- Dipsy can carry out a small set of bounded desktop actions through chat, including browser search, opening safe URLs, opening an existing local path, and focusing or launching supported Windows apps.
- Profile, memory, voice settings, packaging assets, and Windows installer flow are all wired up for a normal packaged desktop release.

## Included behavior

- Floating always-on-top desktop companion window.
- Drag-to-move and double-click chat interaction.
- Persistent local profile and memory storage under `%LOCALAPPDATA%\DipsyDolphin\data\`.
- Supported built-in app targets: browser, Terminal, Explorer, Notepad, and Settings.
- Windows installer with app icon, version metadata, curated release notes, and release smoke validation.

## Notes

- The Windows installer is an online installer and downloads the bundled model and llama.cpp runtime during setup.
- Code signing is not part of `1.0.0`.
