# Prompts

This folder is for reusable AI prompts, planning notes, or generation templates.

Suggested files to add later:

- `feature-template.md` - prompt for adding a new safe feature.
- `refactor-template.md` - prompt for cleanup while preserving behavior.
- `test-template.md` - prompt for generating focused tests.

Suggested prompt shape:

1. State the user-visible goal.
2. Restate the safety boundaries.
3. Name the files likely to change.
4. Define acceptance criteria.
5. Ask for the smallest reasonable implementation.
